Summary: Latitude vs. X

The purpose of this project was to demonstrate mastery of API's and Matplot lib. This was done by analyzing a well understood topic, how weather changes when closer to the equator. For the analysis, data was was pulled from the OpenWeatherMap API to collect data from over 500 randomly selected cities.

After creating the dataset, Matplotlib was used to plot various aspects of weather vs latitude. Factors examined included: temperature, humidity, windspeed, and cloudiness. This site provides the source data, visualizations, and explanations and descriptions of any trends that occured in this analysis.
